Home Page

  • List of accounts includes the name, balance and a link to the Transactions page
  • Bar graph displays a maximum of the last 6 transactions for each account
  • Pie graph displays account assets as a percentage

Flows

  • Home Page Controller loads sessionStorage and displays accounts and graphs, if empty redirects to the Test Page
  • Home Page account links navigate to the Transaction History for the selected account
  • Transfers Page Controller allows transferring funds from one account to another
  • Test Controller allows the user to reset the data, once reset redirects tot he Home Page
